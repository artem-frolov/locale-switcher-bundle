# Locale Switcher Bundle for Symfony 2


## Installation

### Composer

Install bundle by using [Composer](https://getcomposer.org).
Update your project's `composer.json` file to include dependency.

```json
"require": {
    "artem-frolov/locale-switcher-bundle": "~1.0"
}
```

## Usage


    "config": {
